# Geonkick

A kick drum synthesizer.

License: GPLv3

![Screenshot](misc/Screenshot.png)


Status: no release yet

What version 1.0 should include:

[i] - means implemented
[p] - means partial implemented

* Two oscillators:
     - since [i]
     - square [i]
     - triangle [i]
     - sawtooth [i]
     - amplitude envelope [i]
     - frequency envelope [i]
     - low, band, hight pass filter [i]
* Noise:
     - white [i]
     - pink
     - browninan [i]
     - amplitude envelope [i]
     - low, band, hight pass filter [i]
* Kick
     - amplitude envelope [i]
     - low, band, hight pass filter [i]
     - kick length [i]
     - kick limiter [i]
     - Compression [p]
     - Distortion [i]
* Jack support:
     - 1 MIDI in (key velocity sensitive) [i]
     - 2 audio out [i]
* Export to WAVE [i]
     - stereo [i]
     - mono [i]
     - WAV 16. 24. 23 bit [i]
     - FLAC 16, 24 bit [i]
* Save [i]
* Open [i]
* Standalone [i]
* Plugin
     - LV2 [i]
* Platforms:
     - GNU/Linux [i]
